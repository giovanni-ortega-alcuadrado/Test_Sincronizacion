Cupo receptor

El objetivo de esta pantalla es similar al anterior pero en este caso se configura los porcentajes en que puede ser distribuido el portafolio por sucursal, mesa o receptor especifico. Ejemplo En todas las sucursales y todas las mesas se validará que los porcentajes límites no excedan lo siguiente.

•      Acciones           40%

•      Renta fija           40%

•      Repo                 5%

•      Simultaneas      5%

•      TTV                  10%

Para realizar esta configuración en la pantalla se realiza de la siguiente forma.

Esta pantalla se compone de los siguientes campos

Encabezado

•     Sucursal. Este campo corresponde a la sucursal a la que pertenece el receptor.

•     Mesa. Este campo corresponde a la Mesa a la que pertenece el receptor.

•     Código OYD. Cuando aplica muestra el código de cliente en OYD.

Detalle

•     Tipo negocio. Son los diferentes negocios por cada módulo, estos pueden ser: Acciones, Renta fija, Repo, simultánea y TTV.

•     Porcentaje. Valor en porcentaje.
