Cómo aplica la regla “MAQUINA USUARIO”

 

La regla “Maquina usuario” permite validar las maquinas en las cuales puede ingresar ordenes desde OyD plus por usuario. Para configurar esta regla el usuario debe tener maquinas configuradas, si no tiene maquinas configuradas no aplica.

Ejemplo el usuario “oscar.cardona” se le configura la maquina “P-OCARDONAA1”

Y la regla está configurada de la siguiente forma.

Y se ingresa OyD plus para ingresar un orden con el usuario “Oscar. Cardona” como no está en la maquina registrada se debe presentar la validación.
