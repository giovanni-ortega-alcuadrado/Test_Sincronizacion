Mensajes

El objetivo de este maestro es permitir personalizar los mensajes que se visualizan en la aplicación. Por esta razón solo se puede editar el mensaje.

Editar

Mensaje con remplazo: En este campo se puede personalizar el mensaje, se debe tener en cuenta de no quitar las palabras con remplazo que aparecen de la siguiente forma <<Palabra>>; el objetivo de estas palabras es obtener datos de la orden. Ejemplo: El saldo del cliente es <<SALDO>> y el valor ingresado es <<VALOR>>. El cliente no posee el saldo suficiente para crear la orden. En este ejemplo se remplaza <<SALDO>> por el saldo del cliente en el momento que se consulta.
