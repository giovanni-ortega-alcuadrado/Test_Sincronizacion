Ordenes de fondos de Cancelaciones

 

Para realizar un orden de fondos de cancelación se selecciona el tipo producto “Tesorería fondos”, luego se selecciona el cliente, la cartera colectiva o fondo, el encargo  y tipo acción “cancelación”.

Una vez se ingresan los datos del encabezado se procede a crear los detalles o formas de pago de la orden. Las condiciones para ingresar las diferentes son:

•      Debe existir como mínimo una o más formas de pago que el valor corresponda a la suma  del porcentaje máximo de retiro del fondo; es decir que si se tiene el fondo X con saldo disponible de un millón de pesos (1,000,000) y con un porcentaje máximo de retiro del 98%, las formas de pago con valor deben sumar 980,000

•      Una forma de pago con valor en cero.

Ejemplo de ingreso de dos (2) formas de pago cheque con valor y una (1) forma de pago de transferencias con valor en cero.
