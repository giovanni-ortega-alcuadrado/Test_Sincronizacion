Pestaña de Interno

•      Si se selecciona la  pestaña de “Interno” se debe desplegar las siguientes columnas:

o  Tercero = SI/NO. Debe existir siempre.

o  Código = Si es el mismo cliente se escoge de los códigos disponibles de ese cliente. Si no se busca el código del cliente contraparte.

o  Tipo documento. à Si es tercero lo pide. Si no es tercero trae la información del cliente de la orden del encabezado. Y no lo permite cambiar.

o  Numero de documento. à Si es tercero lo pide. Si no es tercero trae la información del cliente de la orden del encabezado. Y no lo permite cambiar. Validar listas restrictivas.

o  Concepto: Combo seleccionable de tblConceptosTesorería. Configurable por Receptor.

o  Descripción del Concepto. Es un texto que el usuario puede digitar adicional. Esto se graba concatenado en la generación del Egreso.

o  Valor: separador de miles y mayor que cero.

o  Tipo GMF. Por encima, por debajo.

o  Valor GMF. Si es un tercero se calcula. Si es el mismo cliente. Se deja en cero. Este campo no es editable (si no está configurado).

o  Valor Neto Cheque = Valor con GMF.
