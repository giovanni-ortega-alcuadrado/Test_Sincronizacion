Control horario

El objetivo de este maestro es control de horario por modulo. Ejemplos

•      Las órdenes de giro con tipo instrucción cheque solo se pueden realizar de 8:00 a 4:30 PM.

•      Las órdenes de giro con tipo instrucción carteras colectivas solo se pueden realizar de 8:00 a 1:30 PM.

•      Las órdenes de bolsa solo se pueden ingresar de 7:00 AM a 5:00 PM.

Cuando se da clic en el botón de Nuevo o Editar. Aparece la siguiente pantalla.

Este maestro se compone de los siguientes campos.

o  Módulo. Actualmente aplica para órdenes y ordenes de tesorería.

o  Tipo negocio. Son los diferentes negocios por cada módulo, estos pueden ser: Acciones, Renta fija, Repo, simultánea y TTV.

o  Tipo orden. Este campo solo aplica para el módulo de órdenes, si la orden es directa o indirecta

o  Tipo producto. Esta es la clasificación de clientes definida por la firma comisionista.

o  Instrucción. Corresponde a la lista de formas de pago del módulo de tesorería.

o  Hora inicio.

o  Hora fin.
