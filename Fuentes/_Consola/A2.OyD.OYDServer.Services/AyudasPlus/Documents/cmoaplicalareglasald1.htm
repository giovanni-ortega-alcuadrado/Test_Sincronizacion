Cómo aplica la regla “SALDO CLIENTE”.

Esta regla permite validar el “saldo de cliente”. Para considerar que una regla se considere activa es porque en el ítem “acción regla” se tiene seleccionado en alguna de estas opciones:

•      No continua (detiene el proceso)

•      Genera advertencia

•      Requiere justificación

•      Requiere aprobación

En este caso está activa porque la acción regla es igual a “Requiere aprobación”

Se ingresa una orden con tipo orden igual a indirecto, tipo de negocio acciones, tipo operación compra, se selecciona el cliente 51960, y se escoge la especie “CEMARGOS”.

En los datos de negocio se  ingresa cantidad 1,000.0000, precio 11,345.0000 y % de comisión 0.300% y calcula valor comisión 34,035.000, IVA 5,445.600 y valor 11,384,480.60.

Se termina de llenar los datos necesarios  para completar la orden (clasificación, forma de pago, ejecución, duración, distribución de comisiones, Datos Leo)
