Pantalla de orden de recibo

El esquema o modelo de pantalla para la orden de recibo es el mismo de Ordenes de giro lo que cambia son los datos que contiene la orden.

Encabezado de la orden

•      Receptor: Campo requerido.

•      Tipo producto. Campo no requerido.

•      Fecha orden: Campo requerido, se pueden crear ordenes futuras y el funcionamiento se realiza de la misma forma que las ordenes de giro.

•      Recibido de: Esta lista puede tomar los siguientes datos Cliente o tercero. Y es campo requerido.

Si se escoge la opción “Cliente”  permite selecciona el código del cliente OYD por la búsqueda de la siguiente forma.

De lo contrario si se escoge “Tercero” se debe digitar el “Nombre”, “Tipo doc.” Y “Nro. Documento”.

Estos datos se deben validar con las listas restrictivas.
