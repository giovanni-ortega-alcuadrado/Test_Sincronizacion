Otros Alcances

1.   Se creara un código asignado a cada tipo de especie en particular con el fin de que los usuarios no requieran registrar toda la información del NEMO y solo con este código el sistema identifique de que especie corresponde. Se quiere que cuando el usuario esté buscando una especie, pueda digitar un nombre conocido para esta especie y que la búsqueda se pueda realizar por este nombre.

Ejemplo: El nemotécnico TFITAV201420 es conocido por los comerciales como “JULIOS20”, entonces lo que se quiere es que cuando el comercial esté ingresando la orden, en el buscador de especies pueda digitar “JULIOS20” y el buscador traiga la especie TFITAV201420.

2.   Los valores denominados en Millones serán separados por un punto con el fin de que el usuario no tenga que registrar todo el valor y el sistema complemente este número. Ejemplo el usuario digita 100. Y el sistema registra 100.000.000, este cambio aplica solo para las pantallas de AV-Ticket y la pantalla de Órdenes.
