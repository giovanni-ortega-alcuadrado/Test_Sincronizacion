Pestaña de Instrucciones

•      Si se selecciona la  pestaña de “Instrucciones” se debe desplegar las siguientes campos:

1.   Forma. Cliente Recoge, Cliente está presente, Llevar a Dirección, Recoge Tercero, Consignar a Cuenta.

2.   Tipo Doc.

3.   Numero Doc.

4.   Nombre Doc.

5.   Dirección Inscrita o No.

6.   Dirección.

7.   Ciudad

8.   Sector.

9.   Entregar a: Tercero, Cliente.

10.  Cuenta Inscrita, Cuenta No Inscrita.

11.  Banco

12.  Tipo de cuenta.

Si 1 es Cliente Recoge. 2, 3, 4 son los datos del cliente y no se cambian. Los otros datos no se piden y no llena nada.

Si 1 es Cliente está presente. 2, 3, 4 son los datos del cliente y no se cambian.

Si 1 es Llevar a dirección. Se pide 5 si es inscripta se selecciona de las del cliente y se llevan a 6, 7, 8 y no se cambian. Si es no inscrita 6, 7, 8 se permiten registrar.  Se pide también 9 si es cliente 2, 3 y 4 se llenan con los datos del cliente si es tercero se deben pedir los datos 2, 3, 4.

SI 1 es recoge tercero pido 2, 3 y 4.

Si 1 es consignar a cuenta se pide 10 si es inscrita se escoge de las cuentas del cliente y se llena 2, 3, 4, 11, 12 sino se pide 2, 3, 4, 11 y 12.

Premisa todos los documentos de identidad y nombres registrados se deben validar en listas restrictivas
