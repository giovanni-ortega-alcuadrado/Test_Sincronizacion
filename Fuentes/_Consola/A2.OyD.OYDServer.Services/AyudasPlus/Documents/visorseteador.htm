Visor seteador

Para el correcto funcionamiento de esta pantalla se debe instalar la aplicación en el escritorio, para realizar esta instalación.

Se da clic “Install A”.consola.SL Application onto this computador…”

Una vez se instala la aplicación estará disponible en el escritorio.

Cuando el usuario ingresa al menú OYD plus à Visor Seteador. Visualizara la siguiente ilustración con las órdenes pendientes.

Esta pantalla debe tener las siguientes características.

•      La pantalla se compone de tres vistas o pestañas.

o  Pendientes: Esta es la vista por defecto cuando se ingresa a esta pantalla y presenta las órdenes en estado “Pendiente”.

o  Procesadas: En esta vista se muestras las órdenes que ya están ejecutadas; es decir las órdenes ejecutadas o procesadas por los ejecutores.

o  Pendientes en bolsa: En esta vista se muestra las órdenes pendientes en la bolsa de valores de Colombia (BVC).

•      El ordenamiento de la lista de datos (grid) es por fecha y hora de orden de forma descendente.
