Generar pago de una operación de bolsa.

 

En la pantalla de generación de Órdenes de Giro el usuario puede consultar las operaciones que se cumplen de un cliente, seleccionar una y generar con esta información la orden de giro.

En el botón de “consultar Liquidaciones “se traen las liquidaciones del cliente con el cual se está realizando la orden de giro; con el objetivo de crear una instrucción con este valor.

 

Si estamos Girando es de las liquidaciones de VENTA, en las ordenes de Recibo es en las Liquidaciones de COMPRA.

 

Se debe tener en cuenta que se debe colocar este botón en todos los detalles (cheque, transferencia, cartera colectiva e interno).

 

 

Este valor sugerido es editable. Si el usuario lo modifica el valor debe quedar claro que cambio realizo.

 

 

Se debe guardar las modificaciones al valor con las correspondientes liquidaciones asociadas.

 

 

En la edición se le resta la nota de bloqueo para tener es saldo real.
