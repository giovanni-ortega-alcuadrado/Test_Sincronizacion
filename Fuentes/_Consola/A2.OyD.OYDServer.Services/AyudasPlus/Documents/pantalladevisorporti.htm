Pantalla de visor por tipo negocio

Una vez se selecciona la orden se debe mostrar la pantalla especifica por tipo de negocio con los siguientes campos.

Tipo de negocio: Títulos de renta variable

Tipo de negocio: Títulos de renta fija

Tipo de negocio: Repos renta variable

Tipo de negocio: Simultanea

Tipo de negocio: TTV

Botones de la pantalla del visor

Botones o acciones que se pueden realizar en el visor

•      Lanzar. Cuándo se presiona  este botón, se realiza el proceso de marcar la orden como lanzada.

•      Lanzar SAE. Este botón debe ser configurable por tipo de negocio; actualmente solo debe habilitarse para acciones. Cuando se da clic en esta opción debe realizar el proceso de lanzar la orden por SAE.

•      Rechazar. Una vez se presiona esta opción se  muestra una lista de causas  de rechazo y una casilla de texto para colocar la observación pertinentes a este.
