Configuración niveles atribución

El objetivo de este maestro es determinar quienes o quien puede aprobar cuando una regla queda pendiente de autorizar. Ejemplo: cuando el cupo de una orden excede el límite determinado se debe recurrir a un nivel o personas que aprueben o rechacen si es el caso el exceso de el limite especificado, en esta pantalla es donde se determinan estos niveles o personas. Como se muestra en esta Ilustración 2, significa que la cuándo se excede el saldo de una orden de tesorería deberá ser aprobada o rechaza por el nivel llamado “Jefe Tesorería”.

Cuando se da clic en el botón de Nuevo o Editar. Aparece la siguiente pantalla.

Esta pantalla se compone de los siguientes campos.

•      Código: Identificador único del maestro. Este dato se crea automáticamente por lo tanto no es modificable.

•      Regla: Se selecciona el código de la regla que se desea configurar.

•      Tipo Documento: El código del tipo de documento matriculado en autorizaciones.

•      Nivel de atribución: Presenta los niveles matriculada para el tipo de documento de autorizaciones.
