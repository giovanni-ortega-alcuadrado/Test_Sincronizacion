Pestaña de Transferencias

•      Si se selecciona la  pestaña de “Transferencia” se debe desplegar las siguientes columnas:

o  Cuenta Registrada (inscrita) = SI/NO. Si es registrada la permite seleccionar de la tabla de tblCuentasCliente, si solo tiene una la trae por defecto.

o  Nro. Cuenta: à Si es inscrita se escoge y se cargan estos datos, no se puede modificar. Si es NO inscrita se debe matricular. .

o  Tipo de Cuenta (ahorros Corriente): à Si es inscrita se escoge y se cargan estos datos, no se puede modificar. Si es NO inscrita se debe matricular.

o  Código Banco: es de la tabla tblBancosNacionales. à Si es inscrita se escoge y se cargan estos datos, no se puede modificar. Si es NO inscrita se debe matricular.

o  Tipo documento Titular. à Si es inscrita se escoge y se cargan estos datos, no se puede modificar. Si es NO inscrita se debe matricular.

o  Numero de documento Titular. à Si es inscrita se escoge y se cargan estos datos, no se puede modificar. Si es NO inscrita se debe matricular.

o  Nombre Titular. à Si es inscrita se escoge y se cargan estos datos, no se puede modificar. Si es NO inscrita se debe matricular.

o  Concepto: Combo seleccionable de tblConceptosTesoreria. Configurable por Receptor.

o  Descripción del Concepto. Es un texto que el usuario puede digitar adicional. Esto se graba concatenado en la generación del Egreso.

o  Valor: separador de miles y mayor que cero.

o  Tipo GMF. Por encima, por debajo.

o  Valor GMF. Si es un tercero se calcula. Si es el mismo cliente. Se deja en cero. Este campo no es editable (si no está configurado).

o  Costo. Aparece por defecto con base en el valor configurado a cada uno de los ítems en la tabla de costos.

o  Valor Neto Cheque = Valor con GMF.
