Lista

Es la primera pantalla que se observa al ingresar a un maestro, esta vista permite observar grupos de registros, con la posibilidad de: seleccionarlos, cambiar el orden de las columnas, ordenar por cada una de las columnas, ir a una página determinada y navegar entre páginas

Reorganizar la Lista:

•      Cambiar el orden de las columnas: Para ordenar las columnas se debe posicionar el cursor en el nombre de la columna que se desea mover, luego se debe hacer clic sostenido y arrastrar, para finalmente soltarla en el lugar deseado

•      Ordenar los registros: para ordenar los registros de forma ascendente se debe hacer un clic en el nombre de la columna por la que se desea hacer el ordenamiento, si se desea ordenar de forma descendente se debe hacer clic dos veces sobre el nombre de la columna

En la parte superior derecha dela lista, se observan los botones que permiten la navegación entre grupos de a diez registros, estos botones son:

•      Primero: Este botón lista los el primer grupo de registros, según el orden en que se encuentran

•      Anterior: Lista los diez registros anteriores, según el orden en que se encuentran

•      Siguiente: Lista los diez registros siguientes, según el orden en que se encuentran

•      Ultimo: Lista el último grupo de registros, según el orden en que se encuentran

•      Paginación: Este campo muestra la página en la que el usuario se encuentra situado del total de páginas de registros, para ir a una página en específico, se debe digitar en el campo de texto el número y luego Presionar la tecla Enter o solamente abandonar el campo de texto.

•      Seleccionar un registro específico: Para seleccionar un registro específico, se debe hacer clic en la fila deseada, sobre la columna cuyos datos aparecen de color azul, de esta manera podremos observar el registro a manera de forma.

Los campos aparecen deshabilitados para proteger la información, si se desea editar el registro se debe hacer clic en el botón Editar.
