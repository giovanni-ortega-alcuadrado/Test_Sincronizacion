Validaciones Modulo de Ordenes de Giro

•      El Cliente no tiene saldo disponible para el giro.

•      El pago se está realizando a un cliente no inscrito.

•      El pago se está realizando a una cuenta NO registrada.

•      El tercero al cual se le gira o se le entrega el cheque está en las listas restrictivas.

•      La carta de orden de giro no está disponible.
