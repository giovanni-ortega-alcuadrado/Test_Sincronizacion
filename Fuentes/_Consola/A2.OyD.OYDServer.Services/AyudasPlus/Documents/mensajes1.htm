Mensajes

El objetivo de este maestro es permitir personalizar los mensajes que se visualizan en la aplicación. Por esta razón solo se puede editar el mensaje.

Cuando se da clic en el botón de  Editar. Aparece la siguiente pantalla.

Solo se permite modifica el ítem “Mensaje con remplazo”

Mensaje con remplazo: En este campo se puede personalizar el mensaje, se debe tener en cuenta de no quitar las palabras con remplazo que aparecen de la siguiente forma <<Palabra>>; el objetivo de estas palabras es obtener datos de la orden.

Ejemplos:

El saldo del cliente es <<SALDO>> y el valor ingresado es <<VALOR>>.

El cliente no posee el saldo suficiente para crear la orden. En este ejemplo se remplaza <<SALDO>> por el saldo del cliente en el momento que se consulta.

Se pueden utilizar palabras como:

•      <<CANTIDAD>>

•      <<CATEGORIACLIENTE>>

•      <<CERTIFICACIONES>>

•      <<dias>>

•      <<DOCUMENTOS>>

•      <<ESPECIE>>

•      <<FECHA>>

•      <<FORMASPAGO>>

•      <<MERCADOESPECIE>>

•      <<NOMBRECLIENTE>> Aplica a clientes relacionados.

•      <<NOMBRECRUZADA>> Aplica a clientes relacionados.

•      <<NOMBRERECEPTOR>>

•      <<PATRIMONIOTECNICO>>

•      <<RELACION>> Aplica a clientes relacionados.

•      <<SALDO>>

•      <<SUMAVALORES>> Aplica para operaciones cruzadas.

•      <<TIPONEGOCIO>>

•      <<TIPONEGOCIO>>

•      <<TIPOOPERACION>>

•      <<TIPOORDEN>>

•      <<TIPOPRODUCTO>>

•      <<TOTALFUTUROREPO>>

•      <<USUARIOOPERADOR>>

•      <<VALOR>>

•      <<VALORFUTUROREPO>>

•      <<VALORINGRESO>> Aplica para operaciones cruzadas.

•      <<VALORORIGINAL>> Aplica para operaciones cruzadas.

•      <<VALORSUPERADO>> Aplica para operaciones cruzadas.
