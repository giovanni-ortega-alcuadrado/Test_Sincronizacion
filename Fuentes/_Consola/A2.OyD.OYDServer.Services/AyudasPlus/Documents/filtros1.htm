Filtros

•      Receptor: Se debe permitir seleccionar todos los receptores o uno específico.

•      Sucursal: Se debe permitir seleccionar todas las sucursales o una especifico. Si se selecciona una sucursal específica ejemplo Medellín en la lista de receptores solo debe aparecer los receptores de la sucursal Medellín.

•      Forma pago. Se debe seleccionar las formas de pago (cheque, transferencia, carteras colectivas e interno).

•      Estado: se debe seleccionar todos los estados o uno específico. Por defecto los documentos pendientes.

•      Fecha orden. Se debe mostrar por defecto la fecha actual.

Para la generación de documentos se debe considerar que el usuario que ingrese a la aplicación y tenga acceso a la pantalla del Tesorero debe tener configurado como mínimo un concepto de tesorería.
