Cómo aplica la regla “PORTAFOLIO CLIENTE”.

 

Esta regla permite validar  que el cliente posea portafolio al momento de la venta. La configuración de la regla en este caso está activa porque la acción regla es igual a “Genera  advertencia”.

Para realizar una orden de venta  para el cliente se debe tener portafolio

Se ingresa el portafolio (custodias) para el cliente “51957”, con la especie “Ecopetrol”

 

De nuevo se procede a ingresar la orden para el cliente “51957” como el cliente y se verifica que muestre el portafolio.
