Configuración Niveles Atribución

El objetivo de este maestro es realizar la relación de la reglas de OYD PLUS con el sistema de Autorizaciones.

Este maestro se compone de los siguientes campos.

•      Código: Identificador único del maestro.

•      Regla: El código de la regla es un valor único que identifica la regla.

•      Tipo Documento: El código del tipo de documento matriculado en autorizaciones.

•      Nivel de atribución: Presenta la estructura matriculada para el tipo de documento de autorizaciones.
