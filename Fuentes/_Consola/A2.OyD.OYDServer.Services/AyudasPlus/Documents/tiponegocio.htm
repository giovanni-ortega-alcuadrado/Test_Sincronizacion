Tipo negocio

El objetivo de este maestro es permitir configurar los diferentes tipo de negocio por especie, tipo producto, certificaciones y documentos clientes.

Cuando se da clic en el botón de Nuevo o Editar. Aparece la siguiente pantalla.

Este maestro se compone de cuatro pestañas o lengüetas

1.   Especies. Permite configurar las especies por tipo de negocio con un valor máximo de negociación.

o  Especie. Se debe tener en cuenta las siguientes características al momento de seleccionar la especie.

•      Si se escoge acciones solo muestra las especies de renta variable activas.

•      Si se escoge renta fija solo muestra las especies de renta fija activas.

•      Si se escoge simultáneas solo muestra las especies de renta fija activas.

•      Si se escoge repos muestra las especies de renta fija y renta variable activas.

o  Si se escoge TTV´s solo muestra las especies de renta variable activas

o  Permitir negociar. SI o NO permite negociar la especie. Ejemplo si se tiene especies todas marcado que se permite negociar y adicionalmente se tiene la especie Fabricato es no está marcada que permita negociar.

Entonces al momento de seleccionar las especies para acciones presenta todas las especies de renta variable activas excepto “Fabricato”.

 

o  Valor máximo negociación. Este valor se utiliza como límite de negociación por orden para la especie seleccionado. Es importante anotar que cuando se deja el valor en cero no se limita la orden.

2.   Tipos producto

o  Tipo producto. Esta es la clasificación de clientes definida por la firma comisionista.

o  Código OYD. Cuando aplica muestra el código de cliente en OYD.

o  Nombre. Nombre del cliente

o  Perfil. Perfil de riesgo del cliente. Ejemplo: Moderado, agresivo, conservador.

o  Valor máximo negociación. Este valor se utiliza como límite de negociación por orden para el tipo de producto seleccionado. Es importante anotar que cuando se deja el valor en cero no se limita la orden.

Ejemplo:

•      Si se selecciona tipo de producto igual a “4. Carteras colectivas” con perfil “Todos” y monto máximo a 100,000

Al momento de realizar la orden solo permite negociar para acciones con los clientes de tipo de producto carteras colectivas con perfil de riesgos todos y con valor máximo de cien mil pesos.

 

3.   Certificaciones. Permite configurar las certificaciones necesarias por tipo de negocio. Se debe tener en cuenta que si se deja en blanco no valida certificaciones.

o  Certificación

Este maestro está relacionado con regla “Certificaciones Receptor” y se valida si la regla esta activa. Para configurar las certificaciones por receptor  se debe realizar por el maestro “configuración receptores” pestaña de certificaciones.

5.   Distribución comisión

o  Tipo producto. Esta es la clasificación de clientes definida por la firma comisionista.

o  Perfil. Perfil de riesgo del cliente. Ejemplo: Moderado, agresivo, conservador.

o  Código OYD. Cuando aplica muestra el código de cliente en OYD.

o  Nombre. Este campo corresponde al Nombre del cliente

o  Límite inferior. Este límite corresponde al inferior valor que tiene la orden.

o  Límite superior. Este límite corresponde al inferior valor que tiene la orden.

o  Distribución en porcentaje. Este dato determina si la comisión se ingresa en porcentaje o valor.

o  Comisión mínima. Comisión mínima que puede tomar la orden.

o  Comisión máxima. Comisión máxima que puede tomar la orden.
