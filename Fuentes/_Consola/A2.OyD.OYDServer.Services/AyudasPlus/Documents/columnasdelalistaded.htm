Columnas de la lista de datos

•      Si se selecciona la forma de pagos “cheques” se debe desplegar:

o  Generar. Este la columna debe permitir marcar o desmarcar una fila; es decir es una columna de check list

o  ID

o  Registrada por. Este dato trae el receptor.

o  Código del cliente OYD.

o  Cliente. Nombre del cliente.

o  Monto.

o  GMF.

o  Neto.

o  Saldo disponible.

o  Beneficiario.

o  Estado.

o  Tipo cheque. Este puede ser gerencia, sencillo, cruzado concatenado con tipo cruce.

o  Datos de la cuenta.

 

•      Si se selecciona la forma de pagos “transferencia” se debe desplegar:

o  Generar. Este la columna debe permitir marcar o desmarcar una fila; es decir es una columna de check list

o  Registrada por. Este dato trae el receptor.

o  Código del cliente OYD.

o  Cliente. Nombre del cliente.

o  Monto.

o  GMF.

o  Neto.

o  Saldo disponible.

o  Cuenta Registrada (inscrita) = SI/NO.

o  Cuenta Registrada (Datos de la cuenta)

o  Nombre 

o  Número documento titular.

o  Beneficiario.

 

•      Si se selecciona la forma de pagos “Carteras Colectivas” se debe desplegar:

o  Generar. Este la columna debe permitir marcar o desmarcar una fila; es decir es una columna de check list

o  Registrada por. Este dato trae el receptor.

o  Código del cliente OYD.

o  Cliente. Nombre del cliente.

o  Monto.

o  GMF.

o  Neto.

o  Saldo disponible.

o  Beneficiario.

o  Nombre de la Cartera Colectiva.

 

•      Si se selecciona la forma de pagos “Interno” se debe desplegar:

o  Generar. Este la columna debe permitir marcar o desmarcar una fila; es decir es una columna de check list

o  Registrada por. Este dato trae el receptor.

o  Código del cliente OYD.

o  Cliente. Nombre del cliente.

o  Monto.

o  GMF.

o  Neto.

o  Saldo disponible.

o  Beneficiario.

Se debe tener en la pantalla un botón para generar documentos. El cual generará los documentos marcados en la lista con los procedimientos que actualmente funcionan en OYD.

Se debe tener en cuenta que cuando se generen documentos de transferencia se debe generar por OYD el comprobante y adicionalmente el ACH.
