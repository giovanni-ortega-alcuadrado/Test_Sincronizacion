Generación Internos

Si se selecciona la forma de pago “Internos” se debe desplegar los detalles de las órdenes que corresponden al receptor, sucursal, forma de pago, estado y fecha de orden seleccionada en los filtros.

Seleccionar todos. Permitirá marcar o desmarcar los documentos a generar.

Columnas disponibles.

•      Generar. Este la columna debe permitir marcar o desmarcar una fila; es decir es una columna de check list

•      ID detalle

•      Id orden tesorería: Orden a la cual pertenece el detalle.

•      Consecutivo. Numero de documento generado en OYD.

•      Registrado por. Este dato trae el receptor.

•      Código cliente.

•      Nombre cliente.

•      Monto.

•      GMF.

•      Valor Giro

•      Beneficiario.

•      Estado.

 

Botón de Generar Documentos

Para generar documento se requiere seleccionar un consecutivo y cuenta bancaria con la cual se genera el documento.

o  Total a generar. Se agregó el total de los detalles seleccionas por forma de pago. Ejemplo: sumatoria de los detalles seleccionados para la forma de pago cheque. Este valor corresponde a la sumatoria del campo valor giro, para este caso es igual a $ 1,073,221.00

o  Saldo banco. Una vez se seleccione la cuenta bancaria se debe mostrar el saldo de cuenta.

o  Consecutivo: Se debe seleccionar los conceptos a los cuales tiene permiso el usuario activo; Es decir que solo muestra los consecutivos

o  Banco: Se trae la información de la cuenta bancaria.

Al dar clic en el botón de Generar Documento se generar las notas de los detalles seleccionado, igualmente se descuenta de la nota de bloqueo el valor de detalle para cada documento generado. Y se visualizan en estado cumplido.
