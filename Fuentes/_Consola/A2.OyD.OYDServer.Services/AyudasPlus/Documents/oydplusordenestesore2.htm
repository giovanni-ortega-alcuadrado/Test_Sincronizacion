La orden de recibo se compone de un encabezado y  sus detalles o formas de recibo.

 Encabezado

1.   Receptor. Presenta la lista de los receptores matriculados al usuario que accede a la aplicación.

2.   Tipo de producto. Para este caso  se selecciona el tipo de producto “Fondos”

3.   Fecha de orden. Determina la fecha de ejecución de la orden.

4.   Recibo de. Aparece por defecto Cliente.

5.   Cartera Colectiva. Lista de carteras colectivas o fondos que posee el cliente seleccionado.

6.   Seleccionar archivo.

Las pestañas que se encuentran disponibles para órdenes de recibo (fondos) son:

•      Cheque

•      Transferencias

•      Consignación

•      Cargar pago a fondos

 

Ejemplo: Se ingresa un orden con el receptor 111, tipo producto igual a “Tesorería Fondos”, fecha orden hoy, recibo de cliente “51962 y cartera colectiva “FONDO RENTA FIJA COLOMBIA”.
