Generación Bloqueo Recursos

 

Si se selecciona la forma de pagos “Bloqueo Recursos” se debe desplegar los detalles de las órdenes que corresponden al receptor, sucursal, forma de pago, estado y fecha de orden seleccionada en los filtros.

Seleccionar todos. Permitirá marcar o desmarcar los documentos a generar.

 

Columnas disponibles.

•      Generar. Este la columna debe permitir marcar o desmarcar una fila; es decir es una columna de check list

•      ID detalle

•      Id orden tesorería: Orden a la cual pertenece el detalle.

•      Consecutivo. Numero de documento generado en OYD.

•      Registrada por. Este dato trae el receptor.

•      Código cliente.

•      Nombre cliente.

•      Motivo Bloqueo.

•      Valor bloqueado

•      Naturaleza.

•      Detalle de bloqueo.

•      Estado.

 

Botón de Generar Documentos

Para generar documento no se requiere seleccionar un consecutivo, ni cuenta bancaria con la cual se genera el documento.

o  Total a generar. Se agregó el total de los detalles seleccionas por forma de pago. Ejemplo: sumatoria de los detalles seleccionados para la forma de pago cheque. Este valor corresponde a la sumatoria del campo valor giro, para este caso es igual a $ 1,073,221.00

o  Saldo banco. No aplica

o  Consecutivo: No aplica. El consecutivo para generar la nota se configura en el maestro de  configurar parámetros, con el nombre de “CONSECUTIVOBLOQRECURSOS”.

o  Banco: No aplica

Al dar clic en el botón de Generar Documento se generar las notas de los detalles seleccionados.

Ejemplo de la nota debito  402237.

Ejemplo de la nota crédito  402238.
